# i4Platform

## It contains:

- `i4Utils`

Universal classes like SyncVar, MutableCharArray and etc.

---

- `i4Framework`

UI Library.
> [!NOTE]
> 
> It requires `i4Utils`.

---

- `i4Framework.swing`
    
An extension to support swing for `i4Framework`.

---

- `i4LCommon`

Classes for my programming language.

> [!CAUTION]
> 
> WIP

---

- `i4LParser`

Parser for my programming language.
> [!NOTE]
>
> It requires `i4LParser`.
